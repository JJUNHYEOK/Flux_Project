# Python Error AI Solver

## Description
- This program solves pip error automatically with GenAI.

## How to use program?
1. create .env file at srcs folder
2. clone the source code.
3. ./srcs/get_err_log_from_pip.py
4. if error occur, select a option that GenAI recommed.

## Requirements
- python 3.11 or higher version
- google-genai
- python-dotenv

## example of .env file
GEMINI_API_KEY=Your GEMINI_API_KEY
