# FLUX : Fast Local Unified eXecution

![flux](pic/flux_ai_removed.png)

## Description
- Flux는 '흐름'이라는 뜻을 가진 단어로, 본 에이전트를 활용하여 개발 흐름이 끊어지지 않도록 합니다.
- **Fast Local Unified eXecution** : 현재 로컬(Local)의 환경을 이해하고, 이를 통합(Unified)된 지능(Gemini + Claude)으로부터 도출된 솔루션을 빠르게(Fast) 실행(eXecution) 가능하도록 제공합니다. 

## How to use program?
1. 라이브러리를 설치할 수 있는 가상환경을 생성하십시오.
2. requirements.txt를 설치하십시오.
3. python main.py pip install <라이브러리명>으로 이 Agent와 함께 개발 환경 구성을 시작하십시오.
4. 걱정은 쓸모없는 것입니다. 오류가 발생하면 이 친구가 다 해결할 것이고, 없다면 평소 일상처럼 흘러갈거에요 !

## Requirements
- python 3.11 or higher version
- google-genai 
- anthropic
- python-dotenv
- rich
